# noknok LED Button

A compact I²C‑controlled button module with integrated LED backlight for the noknok ecosystem.  
Designed for user input, menu navigation, and illuminated UI feedback in modular builds.

![USB Keyboard Button Module](hardware/module-I2C-keyboardbutton-front.png)
![USB Keyboard Button Module](hardware/module-I2C-keyboardbutton-back.png)

---

## Overview

The **Keyboard Button Module** uses a CH32V003 microcontroller to read a tactile push button and control an RGB LED backlight.  
It communicates via the standard noknok **I²C connector**, making it easy to integrate into any project.

Typical use cases:
- Menu selection  
- Confirm / OK / Back buttons  
- Illuminated UI elements  
- Status indication  
- Multi‑button control panels

## Features

- I²C communication (default address: `TBD`)

- Mounting holes for enclosure integration
